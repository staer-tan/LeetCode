## 题目地址
https://leetcode-cn.com/problems/group-anagrams/

## 题目描述
```
给定一个二叉树，返回其节点值的锯齿形层次遍历。（即先从左往右，再从右往左进行下一层遍历，以此类推，层与层之间交替进行）。

例如：
给定二叉树 [3,9,20,null,null,15,7],


Example 1:

    3
   / \
  9  20
    /  \
   15   7

返回锯齿形层次遍历如下：

[
  [3],
  [20,9],
  [15,7]
]
```

## 思路

- 奇偶反转法

思路比较直接清晰，采用广度优先遍历的方法，并设置一个队列。

每扫描到一个节点，将叶子节点放入至队列当中并将其值放入至矢量数组。使用一个标志位flag来记录当前行数

若为偶数（从0层开始）则无需反转，若为奇数则使用reverse函数反转整个数组

时间复杂度O(N)，空间复杂度O(N)

- 双栈

设置两个堆栈的方法，思想更加高级。具体详见代码

时间复杂度O(N)，空间复杂度O(1)

/**
 * Definition for a binary tree node.
 * struct TreeNode {
 *     int val;
 *     TreeNode *left;
 *     TreeNode *right;
 *     TreeNode(int x) : val(x), left(NULL), right(NULL) {}
 * };
 */
class Solution {
public:
    vector<vector<int>> zigzagLevelOrder(TreeNode* root) {
        vector<vector<int>> res;
        if(root == nullptr)
            return res;
        queue<TreeNode*> q;
        q.push(root);
        int flag = 0;
        while(!q.empty())
        {            
            vector<int> temp;
            auto size=q.size();
            while(size--)
            {
                TreeNode* pNode = q.front();
                q.pop();
                temp.push_back(pNode->val);
                
                 if(pNode->left) q.push(pNode->left);
                 if(pNode->right) q.push(pNode->right);
                }
            
            if(flag % 2 == 0)
            {
                res.push_back(temp);
            }
            else
            {
                reverse(temp.begin(), temp.end());
                res.push_back(temp);
            }
            
            flag++;
        }
        
        return res;
        }
};

// 采用堆栈的方法
class Solution {
public:
    stack<TreeNode*> previous,next;
    vector<vector<int>> zigzagLevelOrder(TreeNode* root) {
        vector<vector<int>>result;
        if(!root)return result;
        bool right = false;
        previous.push(root);//手动处理第一层
        result.push_back(vector<int>());
        while(true){
            TreeNode* current = previous.top();previous.pop();
            result.back().push_back(current->val);
            if(right){  //判断右孩子树的情况
                if(current->right)next.push(current->right);
                if(current->left)next.push(current->left);
            }
            else{
                if(current->left)next.push(current->left);
                if(current->right)next.push(current->right);
            }
            if(previous.empty()){
                if(next.empty())return result;//若next空，说明到树底了
                next.swap(previous);//否则，对换两个栈的指针，进入下一层
                right=!right;
                result.push_back(vector<int>());
            }
        }
    }
};
